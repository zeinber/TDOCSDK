# Installation
## Swift Package Manager

1. In Xcode, select **File → Swift Packages → Add Package Dependency...**
2. Enter the repository URL, for example:

   ```text
   https://github.com/zeinber/TDOCSDK.git
   ```
3. Specify a version range (e.g. `from: "1.0.0"`).
